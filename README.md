# 🧠 Disagreement Mapper

> *Find the real crux — not just the two sides*

An AI-powered web tool that analyzes disagreements between two people and breaks them down intelligently — finding common ground, steelmanning both sides, and identifying whether the conflict can be resolved by evidence.

🔗 **[Live Demo](https://disagreement-mapper.vercel.app)**

---

## 💡 What It Does

Most arguments go in circles because people argue positions, not reasons. **Disagreement Mapper** cuts through the noise:

- 🎯 **Finds the real crux** — the actual core of the disagreement
- 🤝 **Identifies common ground** — what both sides actually agree on
- ⚖️ **Steelmans both sides** — gives the strongest version of each argument
- 🔍 **Classifies disagreement type** — factual, values-based, or definitional
- 🧪 **Checks if it's resolvable by evidence** — practical next steps

## 🚀 Live Demo

👉 **[https://disagreement-mapper.vercel.app](https://disagreement-mapper.vercel.app)**

### How to use:
1. Enter your **Gemini API key** (free from [Google AI Studio](https://aistudio.google.com))
2. Type **Side A's** argument
3. Type **Side B's** argument
4. Click **"Map the disagreement"**
5. Get a full AI-powered breakdown instantly!
